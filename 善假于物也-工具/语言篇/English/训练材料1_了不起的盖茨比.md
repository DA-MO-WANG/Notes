```shell
#第一章
在我还年轻还更脆弱的时候，我的父亲给我一些忠告。从那以后，这些话一直萦绕在我的心头。
“当你在批判一个人的时候，要记住！并非这个世界上的人拥有你有的那些条件”。
他没说更多，我们一直以这种沉默的方式来交流。我知道他的意思远不止如此。
因此，我喜欢在生活中保留判断，这种风格成为了习惯。这种习惯导致了我的好奇天性，也让我成为了很多老鸟的受害者。当这种品质出现在正常人身上时，不正常的那些人会很快发现到它，并纠缠在一起。因此这也导致了在大学时期，我被不公正地指责为政治风格的人（耍弄心计的人），因为我暗知陌生人身上这种不文明的隐情。大部分秘密就这么不经意间知道了，我总是认真地装作沉睡或者表现出轻浮的姿态（装糊涂）。这个时候，我意识到我们之间亲密性遭到了动摇的信号。对于年轻男性身上的这种亲密信号，通常被一种压制来破坏。
保留判断（不要轻易下判断），就是保有无尽的希望。我很害怕忘记一些东西，因为这让我感觉我失去了它。
正如我的父亲俯视地建议到，我也自作聪明地重复道：一种根本的行为就是我们出生时并没有被平均的分配。（每个人的行为选择出发点在开始就是不同的）
```

```
在以这种方式自夸到我的包容心后，我承认这种是有限度的。
品行可以建立在坚硬的岩石上，也可以建立在潮湿的沼泽上。然而，我一点也不在意它建立在哪。去年冬天，我从东部回来时，我觉察到我一直想让这个世界整齐划一，符合我的道德视线。我无意带着某种特权以偷窥别人的心灵为乐。
只有盖茨比除外，也就是被冠以书名的男人，他身上包含着我一直鄙视的一切存在。如果个性是一系列成功的姿态，那么他身上自然光彩夺目。他身上存有与生命承诺有关的高度敏感，好像他和一种复杂机器联系在一起，可以获知千里之外的地震。这种反应，与被称为创造性气质的易感受性没有任何关系，它是赋予希望的一种不平凡的礼物。这是一种浪漫的禀赋，我在其他人身上从没看到过的，而且我感觉我再也不会见到了。
盖茨比的结局并非是好的，肮脏的尘土漂浮在它梦醒之处，这也是盖茨比备受折磨的地方，这也让我暂时失去了对人们失败的痛苦和短暂辛劳后愉悦的兴趣。
```

```
我的家族一直显赫，三代以来一直是中西部城市的富裕的人。卡拉伟家族是一个宗族，我们是古老的，祖上是有根的，我们是布克鲁公爵那一支传下来的。但是我们这一脉是我祖父的兄弟创建的，他在五十一岁来到这，是为了给一战做候补兵员，这也就是开始了我父亲今天做五金批发生意。
我从来没有见过我这个好叔公。但是我觉得我应该很像他，特别是在看到挂在父亲办公桌上的那幅画后，我更坚定了这个想法。1915年我也从纽黑文毕业，在我父亲从那毕业25年之后。不久，我就参加了一战。我享受完反攻，以至于后来厌倦了就回来了。中西部并没有成为为世界的温暖中心，贫富分布很不均匀。所以我决定去东部，去学习下债券业务。我认识的每一个人都在这个业务，所以我觉得这个业务应该能够养活我这个单身汉。我的叔叔和婶婶，一直在谈论这件事，就好像为我选择一个预科学校一样。
```
